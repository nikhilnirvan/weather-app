# weather-app
Weather Application | HTML, CSS, JavaScript

Developed a responsive weather web application using HTML, CSS, and JavaScript

Integrated a third-party Weather API to fetch real-time weather data

Implemented search by city name and current location using Geolocation API

Added loading indicators, smooth animations, and user-friendly error handling

Implemented dark mode toggle and saved last searched city using localStorage

Deployed the application using GitHub Pages
Live Demo: https://nikhilnirvan.github.io/weather-app/
